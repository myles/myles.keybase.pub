# [myles.keybase.pub](https://myles.keybase.pub/)

## Usage

**Build** the website:

    $ grunt build

Build the website and **run** a web server:

	$ grunt run

**Deploy** the website to Keybase:

	$ grunt deploy

## Develop

You will need to install the *Ruby*, *Node.js*, and *Bower* libraries:

	$ gem install sass
	$ npm install
	$ bower install
